Byu Testing Center Line Conditions

Google Chrome extension that displays the BYU Testing Center line conditions as well as finals schedule. See website for website version.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to BYU Testing Center Line

This project was built for BYU students to easily check the line conditions at Brigham Young University's testing center. All you have to do is install this free Google Chrome app by going to https://chrome.google.com/webstore/detail/byu-testing-center-line-c/iebcpanhemknjngnemomhddnbgeejofe?hl=en.

Optionally, you may install the app from the source code by downloading it above or by cloning it at Github at https://github.com/bean5/byu-testing-center-line.

If you don't want to install it, you can just go to http://students.cs.byu.edu/~bean5/byu-testing-center-line.git/.
